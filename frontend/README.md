# Employee Attrition Prediction System – Frontend

This project is the frontend interface for the Employee Attrition Prediction System.  
The application allows users to input employee information, send data to the backend API, and display attrition prediction results in a simple and user-friendly interface.

---

## 🚀 Features

- Employee information input form
- Input validation
- Backend API integration
- Attrition probability display
- Risk level classification:
  - High Risk
  - Medium Risk
  - Low Risk
- Loading spinner during prediction
- Error handling for connection and timeout issues
- Simple and clean Streamlit UI

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Requests
- Pandas
